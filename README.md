# Lamp docker microservices stack

## Info
#### Stack:
* nginx:latest
* mysql:5.6
* php:5.6

#### Requirements
* docker
* docker-compose

## Usage

#### Run stack with changes
```
docker-compose build
docker-compose up
```

#### Show containers ip list
```
sh dip.sh
```

#### Remove all containers
```
sh dip.sh
```

#### Remove all images
```
sh dip.sh
```
